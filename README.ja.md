# デザインマンホールマップ

デザインされたマンホールカバーの位置情報をマップ上で確認できるウェブサイトです。

## デモ
[デザインマンホールマップ](https://EiichiMiyagawa.github.io/design-manhole-map/)

## 機能
- デザインマンホールの位置情報の表示
- マンホールの詳細情報の表示（名称、住所、緯度経度、画像）

## データ・API
- [デザインマンホールオープンデータ CSV](https://EiichiMiyagawa.github.io/design-manhole-map/data.csv)
- データソース: [マンホールカード検索 | 下水道広報プラットホーム](https://www.gk-p.jp/mhcard/?pref=18#mhcard_result)
